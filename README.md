# 🎨 SwiftUI Cool Animations

An open-source collection of **beautiful, creative, and modern animations built using SwiftUI**.

---

## 🚀 About the Project

This repository is a growing library of **cool animations in SwiftUI**.  
I’m building this to explore the power of SwiftUI animations and share reusable, easy-to-understand examples with the community.

Whether you're a beginner or an experienced iOS developer, you’ll find something interesting here.

---

## ✨ What to Expect

- Smooth and interactive animations  
- Beginner-friendly examples  
- Reusable SwiftUI components  
- Creative UI/UX ideas  
- Regular updates with new animations
